BX Capybara Game

The capybara game was inspired by the Chrome dino game that appears when there is no internet connection. The game was created by Yuki Kin and Eugenio Animali as part of the CSE104 Web Programming project. Press the "space bar" key to start. Make the capybara jump by using the "up arrow" key, and make it duck by using the "down arrow" key.
